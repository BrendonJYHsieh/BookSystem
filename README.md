# Booking-System
安裝
pip:
下載:https://bootstrap.pypa.io/get-pip.py
```console
python get-pip.py
```
mysql:
```console
pip3 install mysql-connector-python-rf
```
all:
```console
pip install -r requirements.txt
```
執行
```console
python main.py
```
